# 급등주주의보 디자인 아이디어

## 선택된 디자인: 다크 트레이딩 터미널 (Dark Trading Terminal)

**Design Movement:** Bloomberg Terminal meets modern fintech — 데이터 밀도와 가독성을 최우선으로 하는 전문 금융 UI

**Core Principles:**
1. 데이터 우선 — 숫자와 정보가 항상 시각적 중심
2. 긴장감 있는 색상 — 급등(빨강/주황)이 즉각적으로 눈에 띄는 경보 시스템
3. 고밀도 레이아웃 — 스크롤 최소화, 한 화면에 최대 정보
4. 모바일 최적화 — 출퇴근 중 한 손으로 확인 가능

**Color Philosophy:**
- 배경: 짙은 네이비 (#0a0e1a) — 밤새 모니터를 봐도 눈이 편안한 색
- 카드: 반투명 다크 (#111827) with subtle border
- 급등 강조: 선명한 빨강-주황 그라디언트 (#ef4444 → #f97316)
- 보조: 차가운 청록 (#22d3ee) — 중립 정보
- 텍스트: 밝은 회색 (#e5e7eb), 보조 (#6b7280)

**Layout Paradigm:**
- 상단 고정 헤더: 서비스명 + 업데이트 시각 + 국가 필터 버튼
- 카드 그리드: 모바일 1열, 태블릿 2열, 데스크탑 3열
- 각 카드: 순위 배지 + 종목명 + 등락률(크게) + 거래대금 + 자세히 버튼

**Signature Elements:**
1. 등락률 숫자: 굵고 크게, 빨간색 발광 효과 (text-shadow glow)
2. 순위 배지: 1~3위는 금/은/동 색상 강조
3. 자세히 드롭다운: 슬라이드 다운 애니메이션, 격자형 정보 레이아웃

**Typography:**
- 헤더: Bebas Neue (임팩트 있는 디스플레이)
- 숫자/데이터: JetBrains Mono (모노스페이스, 정렬감)
- 본문: Noto Sans KR (한국어 최적화)
